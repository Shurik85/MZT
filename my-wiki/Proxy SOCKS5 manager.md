# 🚀 SOCKS5 (Dante) Manager for Telegram

Удобный Bash-скрипт для быстрой установки и управления **SOCKS5 прокси на Dante** под **Ubuntu** / **Debian**.

Скрипт умеет:

- ⚡ устанавливать и переустанавливать **Dante**
- 👤 создавать основного пользователя `gisproxy`
- 🔐 генерировать случайные пароли в формате `hex`
- ➕ добавлять новых пользователей
- 🗑️ удалять пользователей
- ♻️ менять пароль пользователю
- 🔎 показывать текущие настройки прокси
- 🔗 генерировать готовые **Telegram-ссылки**
- ✏️ открывать конфиг на редактирование
- 🌐 менять порт прокси
- 🛡️ предлагать открыть порт в **UFW**

---

# 📦 Установка в один клик

Самый простой способ — сразу скачать и запустить скрипт одной командой:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/r00t-man/MZT/refs/heads/main/files/Proxy_socks5_dante.sh)
````

> Запускать от `root` или через `sudo -i`

---

# 📥 Установка через скачивание файла

# 🖥️ Что делает скрипт

После запуска откроется меню управления:

```text
1)  Установить / переустановить SOCKS5 (Dante)
2)  Показать текущие настройки прокси
3)  Добавить нового пользователя
4)  Удалить пользователя
5)  Сменить пароль пользователю
6)  Показать ссылку для выбранного пользователя
7)  Удалить всех пользователей кроме gisproxy
8)  Редактировать /etc/danted.conf
9)  Сменить порт прокси
10) Перезапустить danted

0)  Выход
```

---

# ⚙️ Первая установка

После запуска скрипта выбери:

```text
1) Установить / переустановить SOCKS5 (Dante)
```

Далее скрипт предложит выбрать порт:

* `8443` — ✅ рекомендуется
* `9443` — ✅ рекомендуется
* `1194` — если нужен альтернативный порт
* `51820` — если нужен альтернативный порт
* свой порт

После установки скрипт покажет:

* 🌐 IP сервера
* 🔌 порт
* 👤 логин
* 🔐 пароль
* 🔗 готовую ссылку для Telegram

Пример вывода:

```text
=== SOCKS5 (Dante) ready ===
Iface   : ens3
IPv4    : 185.23.19.36
Port    : 8443
User    : gisproxy
Pass    : 8f1b7a6e3c9a2e0d4f7c1a8b6d9e3f4c
Listen  : yes
```

---

# 🔗 Подключение в Telegram

Скрипт автоматически генерирует ссылку вида:

```text
https://t.me/socks?server=185.23.19.36&port=8443&user=gisproxy&pass=PASSWORD
```

### Как подключить:

1. Скопируй ссылку
2. Отправь её себе в Telegram
3. Нажми на неё
4. Telegram сам предложит добавить SOCKS5-прокси

# 🔐 Смена пароля пользователю

# 🧾 Где хранятся данные

### Конфиг Dante

```bash
/etc/danted.conf
```

### Резервная копия конфига

```bash

```

### Локальное хранилище пользователей и паролей

```bash
/root/.dante_socks_users
```

> В этом файле отображаются только пользователи, которые были добавлены именно этим скриптом.

# 🛡️ Firewall / UFW

Если на сервере активен **UFW**, скрипт предложит автоматически открыть выбранный порт для:

* `TCP`
* `UDP`

Если ты отказался при установке, открыть порт можно вручную:

```bash
ufw allow 8443/tcp
ufw allow 8443/udp
```

Заменяй `8443` на свой выбранный порт.

---

# ✅ Проверка работы

Проверить статус сервиса:

```bash
systemctl status danted --no-pager -l
```

Проверить, слушает ли порт:

```bash
ss -tulpn | grep danted
```

Посмотреть логи:

```bash
journalctl -u danted -n 100 --no-pager
```
