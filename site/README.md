# MZT GitHub Pages site

Сайт находится в папке `site/` и построен как набор страниц:

- `index.html` — главная (рендерит корневой `README.md`),
- `section.html` — страница раздела,
- `article.html` — страница отдельной статьи.

## Логика меню и разделов

В верхней полоске меню отображаются только разделы из корня репозитория:

- `Server_Security` → **Настройки сервера**
- `VPN_3x-ui` → **VPN 3x-ui**
- `info` → **Инфо**
- `my-wiki` → **Мои проекты**

Папки `site`, `files`, `help` в меню не включаются.

## Как включить публикацию

1. Откройте `Settings` репозитория на GitHub.
2. Перейдите в `Pages`.
3. Выберите `Deploy from a branch`.
4. Укажите `main` и директорию `/site`.

Итоговый адрес: `https://<owner>.github.io/<repo>/`.
