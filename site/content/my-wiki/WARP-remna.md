## 📄 Инструкция по установке `warp-remnanode.sh`

### 1. **🔧 Загрузите и установите скрипт**

Подразумевается, что Remnanode уже установлен и работает.

Для быстрой установки и запуска скрипта **WARP для Remnanode**, выполните следующую команду:

```bash id="wljo69"
bash <(curl -fsSL https://raw.githubusercontent.com/r00t-man/MZT/main/files/warp-remnanode.sh)
```

### 2. **💡 Что делает скрипт**

Скрипт **warp-remnanode.sh** помогает управлять подключением **Cloudflare WARP** с использованием **SOCKS5** прокси для **Remnanode**. Он выполняет следующие действия:

* 🚀 Устанавливает и настраивает **Cloudflare WARP**.
* 🌐 Настроит прокси-сервер с помощью **SOCKS5** на указанном порту.
* 🔗 Осуществляет подключение и отключение от WARP.
* 🔄 Управляет состоянием сервиса и выполняет перерегистрацию при необходимости.

### 3. **⚙️ Основные функции скрипта**

* **🚀 Установка и подключение WARP**:
  Скрипт автоматически устанавливает **Cloudflare WARP** и настраивает SOCKS5 прокси, чтобы использовать его для подключений.

* **🔄 Изменение порта**:
  Позволяет изменить порт SOCKS5 (по умолчанию `40000`) на любой другой (между 1 и 65535).

* **🔑 Генерация X25519 ключей и shortId**:
  Включает возможность генерировать **X25519 ключи** и **shortId** для использования с **Remnanode**.

* **📊 Отображение информации**:
  Показывает информацию о текущем статусе WARP, IP-адресах, а также настройки outbound и routing для Remnanode.

### 4. **📝 Основные команды**

После запуска скрипта, в меню будут доступны следующие опции:

1. 🟢 Установить / подключить WARP
2. 📑 Показать статус WARP
3. 🔄 Перезапустить WARP
4. 🔴 Отключить WARP
5. 🔁 Перерегистрировать WARP
6. 🔑 Сгенерировать X25519 ключи
7. 🆔 Сгенерировать shortId
8. 🌐 Показать конфигурацию outbound для Remnawave
9. 🛣 Показать правила маршрутизации для Remnawave
10. 📄 Показать полный пример профиля конфигурации
11. 🔄 Перезапустить Remnanode
12. ❌ Выход

### 5. **🔒 Предварительные требования**

* Скрипт требует **root** прав для установки и выполнения.
* Убедитесь, что у вас установлены следующие пакеты:

curl 🌐 — используется для загрузки скриптов и данных с удалённых серверов. <br>
gpg 🔑 — используется для проверки подписей пакетов. <br>
jq 💻 — инструмент для обработки JSON данных. <br>
ss 🔌 (часть пакета iproute2) — утилита для мониторинга сетевых подключений и портов. <br>
 
## ⚠️ Если какие-либо из этих пакетов отсутствуют, скрипт автоматически установит их.
